# My Vocab App

超軽量な英単語学習アプリです。JMdict / KANJIDIC の辞書データ（約9,300語）を使用しています。

https://kouryou-118103.github.io/my-vocab-app/

2025年4月の開発開始から継続的にアップデートを重ねています。
更新履歴は [CHANGELOG.md](./CHANGELOG.md) を参照してください。

## 主な機能

- 4択クイズ（選択肢数のカスタマイズ可）
- フラッシュカードモード
- 単語入力モード（日本語→英語）
- 記録モード（ミスした単語をCSVで出力、復習に利用可能）
- 「いやらしいモード」（ABC順で近い紛らわしい単語を選択肢に出す）
- ダークモード
- ショートカットキー対応

## 技術的なポイント

- フレームワーク不使用（バニラJS / HTML / CSS）
- 大規模な単語データ（9,000語超）でも高速に動作するよう継続的に最適化
